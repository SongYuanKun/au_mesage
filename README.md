# AU Metal Price Platform

Language / 语言：

- [中文 README](./README.zh-CN.md)
- [English README](./README.en.md)

This repository contains a lightweight precious metal price collection & analysis platform (Gold/Silver): multi-source collectors → MySQL storage → Flask web pages + REST APIs, plus optional notification channels (WeCom/Telegram/Email).

本仓库是一个轻量级贵金属价格采集与分析平台：多源采集 → MySQL 入库 → Flask 页面 + REST API，并支持企业微信/Telegram/邮件通知（可选）。
